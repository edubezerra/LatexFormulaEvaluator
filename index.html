Welcome to LatexFormulaEvaluator Pages.

This project is a Java application to evaluate and parse numerical formulas writen in the LaTeX syntax. Currently, only a limited subset of expressions is recognized. To see examples of recognazable formulas, read the class named LatexFormulaEvaluatorTest, which is a test class for the JUnit framework.
